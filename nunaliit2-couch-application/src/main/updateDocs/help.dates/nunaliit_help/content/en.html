Dates can be expressed using one of the following formats:

  • YYYY
  • YYYY-MM
  • YYYY-MM-DD
  • YYYY-MM-DD hh:mm
  • YYYY-MM-DD hh:mm:ss

In Nunaliit, all dates are recorded as time intervals. If only a year is specified, then the recorded time interval is a period that starts from the first day of the year until the last day of the year. Dates recorded with a greater precision yield shorter time intervals.

Time intervals that do not coincide with year or month boundary can be expressed as a duration between two dates. A duration is expressed as: 

		<start-date> / <end-date>

If a duration is continuing until the present (e.g. employed since), then the duration can be entered using the following format: 

		<start-date> / -

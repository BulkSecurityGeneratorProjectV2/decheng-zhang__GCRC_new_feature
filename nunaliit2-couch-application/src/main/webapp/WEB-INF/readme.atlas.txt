Atlas Project
=============

This project provides an infrastructure to develop atlas instances
using a CouchDb database.


atlas_couchapp
--------------

CouchApp application containing all supporting files for an atlas
hosted by CouchDb


atlas_install
-------------

Set of scripts used to install an instance of an atlas.


atlas_cron
-------------

Set of scripts used in a daily CRON job to update the servlet
and atlas instance.


config_couchapp
---------------

CouchApp application using the files needed to run a database that
contains configuration information.
